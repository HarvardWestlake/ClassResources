Trigonometric Substitution (Calculus C)

Trigonometric substitution is an extension of your integration toolkit. It targets a specific class of integrals whose algebraic structure suggests a trig change of variables—typically when a quadratic expression appears under a square root or in a denominator. Use this widget to choose the right substitution, build a reference triangle, and carry the integral through to a clean back‑substituted result.

Quick Reference

Patterns
  • √(a² − x²) → x = a sin θ
  • √(a² + x²) → x = a tan θ
  • √(x² − a²) → x = a sec θ
Identities
  • 1 − sin²θ = cos²θ
  • 1 + tan²θ = sec²θ
  • sec²θ − 1 = tan²θ

How to Use This Widget — Step-by-Step Guide

1. Selecting a Function

Use the Function dropdown to choose which trigonometric substitution pattern you want to explore. Available options include:

  • Square root functions: sqrt(a² - x²), sqrt(a² + x²), sqrt(x² - a²)
  • Inverse square root functions: 1/sqrt(a² - x²), 1/sqrt(a² + x²), 1/sqrt(x² - a²)
  • Rational functions: 1/(a² - x²), 1/(a² + x²), 1/(x² - a²)
2. Adjusting the Parameter 'a'

The a input controls the shape of the function. When you change 'a', the function curve updates automatically. The bounds (L and R) remain unchanged unless they become invalid for the new function domain.

3. Setting Integration Bounds

Enter values for Left bound L and Right bound R to define your integration interval. You can:

  • Type values directly into the input fields
  • Drag the circular handles on the graph to adjust bounds visually
  • The shaded area shows the region being integrated
  • The approximate area value appears below the graph
4. Show Trig Substitution Checkbox

When checked, displays a panel below the graph showing:

  • Mapping: The substitution formulas (x = ..., dx = ..., y = ...)
  • θ-bounds and integrand: The transformed integral in terms of θ

This helps you see how the integral transforms from x-space to θ-space.

5. Show Geometry Checkbox

When enabled, overlays reference geometry on the graph to visualize the trigonometric substitution:

  • For sin substitutions: Shows a reference circle and right triangle with angle θ labeled
  • For tan substitutions: Shows a reference right triangle demonstrating the relationship
  • For sec substitutions: Shows a reference right triangle for the secant substitution

The geometry helps you understand how to back-substitute after integrating in terms of θ. Note: Geometry only appears when bounds are set (not in indefinite integral mode).

6. Indefinite Integral Checkbox

When checked:

  • Hides the bounds input fields (L and R)
  • Removes the shaded area from the graph
  • Shows only the function curve
  • Updates the trig substitution panel to show the indefinite integral form
  • Disables geometry overlay (since it requires bounds)

Use this mode when you want to explore the general form of the integral without specific bounds.

7. Collapse Button

The Collapse button in the top-right corner of the widget allows you to:

  • Hide/show the entire widget content
  • Save screen space when you're not actively using the widget
  • Your preference is saved and will persist when you reload the page

Click "Collapse" to hide, or "Expand" to show the widget again.

8. Copy Shareable Link Button

Click Copy shareable link to copy a URL that includes all your current settings:

  • Selected function type
  • Parameter 'a' value
  • Integration bounds (L and R)
  • Checkbox states (Show trig substitution, Show geometry, Indefinite integral)

Share this link with others or bookmark it to return to the same configuration.

9. Interacting with the Graph

The graph is interactive:

  • Drag the circular handles on the x-axis to adjust integration bounds
  • Vertical dashed lines show your current bounds
  • Shaded area represents the region being integrated
  • Grid lines help you read approximate values
  • Axis labels show the scale
10. Reading the Output

Below the graph, you'll see:

  • Area estimate: The approximate numerical value of the definite integral
  • Trig Substitution panel: (if enabled) Shows the mathematical transformation
  • Metrics box: Displays the integration interval and calculated area

How to Solve a Trig Substitution

  1. 1
    Read the yellow Quick Reference box and match your pattern to choose the substitution (sin / tan / sec).
  2. 2
    Identify the trig identity to simplify to θ (e.g., 1 − sin²θ = cos²θ, 1 + tan²θ = sec²θ).
  3. 3
    Integrate in θ, then use the reference triangle to back‑sub and express the answer in x.
