Getting started

Use this notebook while playing the Mineral Identification Game. Start with simple tests (scratch test for hardness, magnetism, acid reactivity), then inspect luster and cleavage. Use search and filters in the directory to quickly find candidates and compare properties.

Property glossary

Hardness (Mohs)

Resistance to scratching. Reference tools: Fingernail ≈2.5, Copper ≈3.0, Glass ≈5.5, Steel ≈6.5.

Luster

Quality of reflected light. Common types here: metallic, submetallic, vitreous, pearly, resinous, earthy.

Cleavage

Tendency to break along flat planes; shown here as number of visible cleavage planes.

Magnetism

None, weak, or strong attraction to a magnet.

Acid reactivity

Reaction to cold dilute HCl: strong fizz, weak fizz (powdered), or none.
